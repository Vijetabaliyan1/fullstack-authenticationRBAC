# Fullstack Authentication System with RBAC

## 📌 Project Description
This project is a full-stack authentication system built using React, Node.js, Express, and MongoDB. It implements secure login functionality, JWT-based authentication, protected routes, and role-based access control (RBAC).

## 🚀 Features
- User Login with validation
- JWT Authentication
- Protected Routes
- Role-Based Access Control (Admin & User)
- Admin Dashboard
- User Dashboard
- Secure API handling

## 🛠️ Tech Stack
- Frontend: React.js, React Router, Material UI
- Backend: Node.js, Express.js
- Database: MongoDB
- Authentication: JWT

## 🔐 Roles
- Admin: Can access admin dashboard
- User: Limited access (dashboard only)
